# Визуализация графов

Приложение для визуализации графов на основе корреляционных матриц из Excel файлов.

## Функциональность
- Загрузка данных из Excel файлов
- Автоматическое вычисление корреляционной матрицы
- Построение графа в 2D и 3D
- Визуализация весов связей

## Установка и запуск
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите приложение: `python app.py`

## Использование
1. Нажмите "Выбрать данные" и выберите Excel файл
2. Выберите тип визуализации (2D или 3D)
3. Нажмите "Построить граф"
